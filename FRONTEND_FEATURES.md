## 🎯 使用指南

### 1. 创建聊天室

1. 点击聊天室区域的 **➕** 按钮
2. 填写聊天室信息：
   - **名称**: 给聊天室起个有趣的名字
   - **描述**: 简单描述聊天室的用途
   - **隐身模式**: 勾选后用户对AI不可见
   - **用户描述**: 告诉AI你是谁（仅非隐身模式）

**示例配置：**

**普通聊天室：**
```
名称: AI哲学讨论
描述: 探讨人工智能与哲学的深度话题
隐身模式: ❌ 关闭
用户描述: 我是一个对AI伦理很感兴趣的哲学学者
```

**隐身观察室：**
```
名称: AI自然对话观察
描述: 观察AI角色之间的自然交流
隐身模式: ✅ 开启
用户描述: (自动隐藏)
```

### 2. 切换聊天室

1. 使用聊天室下拉菜单选择不同的聊天室
2. 系统会自动：
   - 清空当前聊天记录
   - 重新连接WebSocket
   - 加载新聊天室的角色和消息
   - 显示切换成功通知

### 3. 管理聊天室设置

1. 选择聊天室后，点击 **⚙️ 设置** 按钮
2. 可以修改：
   - 聊天室名称和描述
   - 隐身模式开关
   - 用户自我描述
3. 查看聊天室统计：
   - 当前角色数量
   - 历史消息数量

## 🎨 界面特色

### 视觉指示器

- **👥 普通模式**: 绿色指示器，用户可参与对话
- **🥷 隐身模式**: 灰色指示器，用户隐身观察
- **角色数量**: 显示在聊天室名称后，如 `(3角色)`

### 智能表单

- **条件显示**: 隐身模式开启时自动隐藏用户描述字段
- **实时验证**: 表单字段实时验证和提示
- **友好提示**: 每个功能都有详细的说明文字

### 响应式通知

- **成功操作**: 绿色通知，如"聊天室创建成功！"
- **信息提示**: 蓝色通知，如"已切换到聊天室: xxx"
- **错误警告**: 红色通知，显示具体错误信息
- **自动消失**: 3秒后自动消失，不干扰用户操作

## 🔄 工作流程

### 典型使用流程

1. **启动应用** → 访问 http://localhost:8000
2. **配置API** → 点击"配置API"设置AI服务
3. **创建聊天室** → 根据需求创建不同类型的聊天室
4. **添加AI角色** → 为聊天室添加有趣的AI角色
5. **开始聊天** → 启动自动聊天或手动发送消息
6. **切换体验** → 在不同聊天室间切换，体验不同模式

### 隐身模式体验流程

1. **创建隐身聊天室** → 开启隐身模式
2. **添加多个AI角色** → 不同性格的AI角色
3. **启动自动聊天** → 观察AI之间的自然对话
4. **静默观察** → AI无法感知用户存在
5. **纯净体验** → 享受AI角色间的纯粹交流

### 个性化助手流程

1. **创建个人聊天室** → 关闭隐身模式
2. **设置用户描述** → 详细描述自己的身份和需求
3. **添加专业AI角色** → 如编程助手、写作顾问等
4. **个性化对话** → AI根据用户描述调整回复风格
5. **一对一深度交流** → 获得个性化的AI服务
